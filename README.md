# shell

Simple shell project - exercise from book
"Modern Operating Systems", Andrew S.Tanenbaum

Based on assignment:
http://www.cs.cornell.edu/Courses/cs414/2004su/homework/shell/shell.html

###Include:

  - history,
  - pipes,
  - arrow-searching in history
